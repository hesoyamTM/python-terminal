# Простой командный интерпретатор (Shell)

## Описание проекта

Этот проект представляет собой простой командный интерпретатор (shell), реализующий основные команды для работы с файловой системой с дополнительными возможностями.

## Функциональность

    1. Команда ls
        Отображение списка файлов и каталогов в текущем рабочем каталоге

    2. Команда cd
        Переход в указанный каталог

    3. Команда cat
        Вывод содержимого указанного файла в консоль

    4. Команда cp
        Копирование файла из источника в назначение

    5. Команда mv
        Перемещение или переименование файла/каталога

    6. Команда rm
        Удаление указанного файлас и каталога

    7. Команда zip
        Создание архива формата ZIP из каталога

    8. Команда unzip
        Распаковка архива ZIP в текущий каталог

    9. Команда tar
        Создание архива формата TAR.GZ из каталога

    10. Команда untar
        Распаковка архива TAR.GZ в текущий каталог

    11. Команда grep
        Поиск строк, соответствующих шаблону pattern в файлах

    12. Команда history
        Вывод последних введённых команд

    13. Команда undo
        Отмена последней команды из списка cp, mv, rm

## Использование

```bash

# Запуск интерпретатора

task run

# Или

python3 -m src.main

# Примеры команд в интерпретаторе

ls
ls -1 /path/to/directory
cd ~
cd ..
cat filename.txt
cp source.txt destination.txt
cp -r source_dir destination_dir
mv oldname.txt newname.txt
rm file.txt
rm -r directory
```

## Файлы системы

    - shell.log — файл логов всех операций

    - .history — файл истории команд

    - .cancelable_history — файл истории команд с отменой последней команды

    - .trash — временная корзина для восстановления удалённых файлов

## Разработка

    - Проект разработан по чистой архитектуре с использованием модулей и классов

    - Использован loguru для логирования всех операций

    - Использован pytest для тестирования кода

### Слой представления

    Представляет из себя бесконечный цикл команд и вывода результатов в терминал.

### Слой бизнес-логики

    Содержит в себе парсер команд и обработку их действий.
    В терминале находится словарь, который по названию команды выдает функцию для её обработки.
    Каждая команда принимает список аргументов и возвращает результат выполнения команды.

### Слой адаптеров

    Адаптеры представляют собой интерфейсы для работы с файловой системой
